WP Idea Stream addon to share ideas within a BuddyPress powered community

== Description ==

Since version 2.4.0 of WP Idea Stream BuddyPress intergration is no more built-in the plugin but is now separately available from this addon. It requires BuddyPress 2.5+ and of course WP Idea Stream 2.4+.

Once activated, the WP Idea Stream profile will be moved as a new BuddyPress user's profile subitem and if you enabled the BuddyPress User Groups, Group members will be able to share their ideas within the Group.

== Installation ==

You can download and install BP Idea Stream using the built in WordPress plugin installer. If you download BP Idea Stream manually, make sure it is uploaded to "/wp-content/plugins/bp-idea-stream/".

Activate BP Idea Stream in the "Plugins" admin panel using the "Activate" link.
